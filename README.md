# Java Core Framework
Java Core Framework

## 1. JCF 개요
+ JCF (Java Core Framework) 는 대우정보시스템의 Java 개발 프레임워크
+ J2EE 기반의 어플리케이션 개발을 위한 재사용 가능한 자사 표준 프레임워크
+ 애플리케이션 개발에 필요한 실행서비스 , 공통서비스 그리고 개발서비스로 구성 된 통합 개발플랫폼을 제공
+ 다년간의 프로젝트 적용을 통해 안전성 및 품질 검증

### 1.1 개발 정보
+ 개발시기: 2004 ~ 2011년 초
+ 개발참여: 대우정보시스템 프레임워크센터
